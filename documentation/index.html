Documentation

Theme

All themes (materials) are enum cases from NSVisualEffectMaterial. This option only works on macOS 10.11+.

Preferences

All preferences related to playback are achieved by passing options to mpv. See MpvController.swift in source code for more information.

Keybindings

Currently, IINA does not have a user-friendly key binding setting interface. When user add a new key binding, it can record keys but user must input the corresponding command manually. For configurations, it only parses each configuration as a conf file and save it (Note that after saving, all comments will lost).

For best compatibility, it interprets user's key input at runtime and pass them to mpv directly using keydown command, hence all menu shortcuts (like Cmd+V) will not be captured. Of course, some prefixes like no-osd will not work unless enabling "Use MPV's OSD" in "Advanced" panel.

Advanced

Advanced settings are only for mpv users who have basic knowledge on mpv's option and configuration system. All settings in advanced tab will take effect after restarting application.

Enable logging

This will pass --log-file=<path> to mpv. User cannot change the default logging path, but overriding such option in "additional MPV options" should work. The default logging directory can be revealed in finder by clicking "Open log directory" button.

Use MPV's OSD

Normally, IINA will pass --osd-level=0 to totally disable internal OSD, and use its own OSD system instead. However there still exists lots of problems in current system, so user may want to use mpv's OSD for more information, or for compatibility of previous scripts. Enabling this option will disable IINA's OSD and set --osd-level to 1 by default.

Config directory

MPV Client API will not automatically load ~/.config/mpv as config Directory. If you are a previous mpv user and wants to keep all settings and scripts, you may enable this option. This is equivalent of passing --config=yes --config-dir=<dir> to mpv.

Additional MPV options

You can add additional MPV options in this section. All options here has higher priority than all other options in preferences, including those in "Advanced" panel. The following list shows priority of options (low to high):

  • Options translated from user preferences
  • Options in config files from "Config directory" (if enabled)
  • Options in "Additional MPV options"
  • Some special options, namely --vo for stability
